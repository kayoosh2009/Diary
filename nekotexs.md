# Разаработка Nekotexs
Идея прроекта:
Nekotexs это токен в сети Solana который предназначен для болаготворительности бездомным котам во всем мире, с каждой транзакции берется 2-3% и переносится на кошелек Solana который специально создан для накопления баланса который потом будет пожертвован котам во всем мире. <br>
Сам по себе это милый неко проект который обьединияет всех пользователей с одной целью помомчь котикам во всем мире. <br>
